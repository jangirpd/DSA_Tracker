## What is DSA Tracker 🤔

#### DSA Tracker is a comprehensive list of 400 + topic wise questions to build your confidence in data structure and algorithms and prepare yourself for placements.

#### DSA Tracker doesn't guarantee a job but guarantees your confidence in solving any coding problem if done in the right way 👍🏻.
